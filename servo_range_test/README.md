# Servo Range Test

এই স্ক্রিপ্ট দিয়ে আপনার SG90 সার্ভো মোটরের নিরাপদ রেঞ্জ টেস্ট করতে পারবেন।

## কিভাবে ব্যবহার করবেন

### ১. Arduino IDE তে খুলুন
- Arduino IDE খুলুন
- `servo_range_test.ino` ফাইল খুলুন
- ESP32 বোর্ড সিলেক্ট করুন

### ২. আপলোড করুন
- Upload বাটনে ক্লিক করুন
- Serial Monitor খুলুন (Baud Rate: 115200)

### ৩. পর্যবেক্ষণ করুন
টেস্ট চলাকালীন দেখুন:
- ✅ মোটর মসৃণভাবে ঘুরছে কিনা
- ✅ কোনো বিপ/গুঁজগুঁজ শব্দ হচ্ছে কিনা
- ✅ কোনো এঙ্গেলে আটকে যাচ্ছে কিনা

## যদি সমস্যা হয়

যদি এখনও 160° এ সমস্যা হয়:
1. `servo_range_test.ino` খুলুন
2. `SERVO_MAX_ANGLE` এর মান 150 করুন
3. আবার আপলোড করুন

## মূল প্রজেক্টে ফিরে যান

টেস্ট শেষ হলে:
1. `Main` ফোল্ডারের `Main.ino` খুলুন
2. আপলোড করুন
3. রোবট এখন নিরাপদভাবে কাজ করবে
